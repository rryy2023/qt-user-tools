# 标题颜色优化说明

## 🎯 问题

"预览图问题"等标题在白色背景上看不清楚，需要优化颜色对比度。

## ✅ 优化方案

### 1. 标题颜色调整

**优化前**：
- 普通问题标题：`#1a1a1a`（深灰色）
- 在某些情况下可能不够清晰

**优化后**：
- 普通问题标题：`#000000`（纯黑色）
- 关键问题标题：`#ff4d4f`（红色，保持不变）
- 确保在白色背景上清晰可见

### 2. 描述文字颜色

- 统一使用：`#333333`（深灰色）
- 确保在白色背景上清晰可见

### 3. 样式优先级

- 使用内联样式确保颜色不被全局样式表覆盖
- 在 `apply_style()` 方法中再次设置，确保优先级

## 📋 修改内容

### `gui/widgets/problem_card.py`

1. **标题颜色**：
   ```python
   # 关键问题使用红色，普通问题使用深黑色
   title_color = "#ff4d4f" if is_critical else "#000000"
   ```

2. **样式表增强**：
   ```python
   # 确保标题和描述颜色不被样式表覆盖
   if is_critical:
       self.title_label.setStyleSheet("color: #ff4d4f; font-weight: bold;")
   else:
       self.title_label.setStyleSheet("color: #000000; font-weight: bold;")
   self.desc_label.setStyleSheet("color: #333333;")
   ```

3. **全局样式**：
   ```css
   ProblemCard QLabel {
       background: transparent;
       color: #000000;  /* 默认黑色 */
   }
   ```

## 🎨 颜色方案

| 元素 | 普通问题 | 关键问题 |
|------|----------|----------|
| 标题 | `#000000` (黑色) | `#ff4d4f` (红色) |
| 描述 | `#333333` (深灰) | `#333333` (深灰) |
| 背景 | `#ffffff` (白色) | `#fff1f0` (浅红) |
| 边框 | `#e0e0e0` (浅灰) | `#ffccc7` (浅红) |

## ✨ 效果

- ✅ 标题在白色背景上清晰可见
- ✅ 关键问题使用红色突出显示
- ✅ 描述文字对比度足够
- ✅ 整体可读性大幅提升

## 📝 注意事项

1. **颜色优先级**：内联样式优先级最高，确保不被覆盖
2. **关键问题**：保持红色以突出重要性
3. **可访问性**：黑色文字在白色背景上对比度最高（21:1）
