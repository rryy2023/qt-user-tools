# 样式调整说明

## 问题描述

用户反馈：文字和背景看不清，需要调整样式并重新打包。

## 已完成的样式调整

### 1. 文字颜色对比度优化

#### 问题卡片（ProblemCard）
- **标题文字**：从默认颜色改为 `#1a1a1a`（深色），提高对比度
- **描述文字**：从 `#666`（浅灰）改为 `#333333`（深灰），字体从 9px 增大到 10px
- **关键问题标题**：使用 `#ff4d4f`（红色）突出显示
- **字体大小**：标题从 12px 增大到 13px，描述从 9px 增大到 10px

#### 主窗口（MainWindow）
- **标题文字**：使用 `#1a1a1a`（深色）
- **版本信息**：从 `#999` 改为 `#666666`，提高可读性
- **状态栏文字**：从 `#666` 改为 `#333333`，字体加粗（font-weight: 500）

### 2. 背景颜色调整

#### 主窗口背景
- 从 `#f5f5f5`（浅灰）改为 `#ffffff`（纯白），提供更好的文字对比度

#### 问题卡片背景
- **普通问题**：使用白色背景 `#ffffff`，边框 `#e0e0e0`
- **关键问题**：使用浅红色背景 `#fff1f0`，边框 `#ffccc7`，突出重要性
- **悬停效果**：普通问题 `#fafafa`，关键问题 `#ffe7e5`

### 3. 按钮样式优化

- **按钮文字**：使用白色 `#ffffff`，背景蓝色 `#1890ff`
- **悬停效果**：背景变为 `#40a9ff`
- **字体大小**：11px，加粗显示

### 4. 边框和视觉层次

- 所有卡片添加 2px 边框，提高视觉分离
- 悬停时边框变为蓝色 `#1890ff`，提供交互反馈

## 修改的文件

1. **gui/widgets/problem_card.py**
   - 优化文字颜色和字体大小
   - 调整背景色方案（普通问题白色，关键问题浅红色）
   - 关键问题标题使用红色突出显示

2. **gui/main_window.py**
   - 优化标题和标签文字颜色
   - 调整状态栏文字对比度

3. **gui/resources/styles.qss**
   - 主窗口背景改为纯白色

## 注意事项

- 所有文字现在使用深色（#1a1a1a 或 #333333），在白色背景上对比度充足
- 关键问题（主站打不开、Safari无法打开）使用红色标题和浅红色背景突出显示
- 按钮使用蓝色背景和白色文字，对比度良好
